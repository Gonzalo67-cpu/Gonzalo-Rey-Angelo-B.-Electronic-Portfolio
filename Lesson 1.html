What is CSS?

CSS stands for Cascading Style Sheets.

CSS describes how HTML elements are displayed on screen, paper, or in other media.

CSS saves a lot of work. It can control the layout of multiple web pages all at once.

External stylesheets are stored in CSS files.

Why use CSS?

  • CSS defines styles for web pages, including design, layout and variations for different devices.
  • Provides consistency across web pages.
  • Saves time: one CSS file can control multiple pages.
  • Improves accessibility and responsiveness.

Layers of a Webpage

  1. Behavior Layer
  2. Presentation Layer
  3. Content Layer

Content Layer

  • Defines and lays out text, images, animation, sound, video, and everything you want to put in a web page using tags.
  • Forms the underlying HTML structure — like the skeleton of a house.

Presentation Layer

  • Defines how people see your webpage in a browser.
  • Ensures text is readable and site remains usable when zoomed.
  • Keeps navigation consistent and organized.

Behavior Layer

  • Handles user interactions and responses.
  • Ensures all functionality is accessible to users.

Benefits of CSS

  • CSS saves a lot of work.
  • Allows easier updates and consistent styling.
  • Separates structure (HTML) and presentation (CSS).

Capabilities of CSS

  1. Quick updates across pages.
  2. Precise positioning of elements.
  3. Layering objects (3D positioning).
  4. Creating reusable custom tags and styles.

Advantages of using CSS

  • Saves development time.
  • Reduces file size and improves loading speed.
  • Improves design flexibility and maintenance.

Prerequisites

Basic knowledge of HTML or XHTML is needed to learn CSS.
