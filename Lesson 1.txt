      <hr>

      <h3>What is CSS?</h3>
      <p><strong>CSS</strong> stands for Cascading Style Sheets.</p>
      <p>CSS describes how HTML elements are displayed on screen, paper, or in other media.</p>
      <p>CSS saves a lot of work. It can control the layout of multiple web pages all at once.</p>
      <p>External stylesheets are stored in CSS files.</p>

      <h3>Why use CSS?</h3>
      <ul>
        <li>CSS defines styles for web pages, including design, layout and variations for different devices.</li>
        <li>Provides consistency across web pages.</li>
        <li>Saves time: one CSS file can control multiple pages.</li>
        <li>Improves accessibility and responsiveness.</li>
      </ul>

      <h3>Layers of a Webpage</h3>
      <ol>
        <li>Behavior Layer</li>
        <li>Presentation Layer</li>
        <li>Content Layer</li>
      </ol>

      <h4>Content Layer</h4>
      <ul>
        <li>Defines and lays out text, images, animation, sound, video, and everything you want to put in a web page using tags.</li>
        <li>Forms the underlying HTML structure — like the skeleton of a house.</li>
      </ul>

      <h4>Presentation Layer</h4>
      <ul>
        <li>Defines how people see your webpage in a browser.</li>
        <li>Ensures text is readable and site remains usable when zoomed.</li>
        <li>Keeps navigation consistent and organized.</li>
      </ul>

      <h4>Behavior Layer</h4>
      <ul>
        <li>Handles user interactions and responses.</li>
        <li>Ensures all functionality is accessible to users.</li>
      </ul>

      <h3>Benefits of CSS</h3>
      <ul>
        <li>CSS saves a lot of work.</li>
        <li>Allows easier updates and consistent styling.</li>
        <li>Separates structure (HTML) and presentation (CSS).</li>
      </ul>

      <h3>Capabilities of CSS</h3>
      <ol>
        <li>Quick updates across pages.</li>
        <li>Precise positioning of elements.</li>
        <li>Layering objects (3D positioning).</li>
        <li>Creating reusable custom tags and styles.</li>
      </ol>

      <h3>Advantages of using CSS</h3>
      <ul>
        <li>Saves development time.</li>
        <li>Reduces file size and improves loading speed.</li>
        <li>Improves design flexibility and maintenance.</li>
      </ul>

      <h3>Prerequisites</h3>
      <p>Basic knowledge of HTML or XHTML is needed to learn CSS.</p>
